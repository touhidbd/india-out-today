== Description ==

India Out Today is a WordPress plugin that helps users boycott products and find alternative options.

= Plugin Features =

* Search – Easily find the Indian products
* Filter – Easily filter the categories
* Alternative – Easily find the alternative brand and products

== API Provider ==
* [Indiaout.Today](https://indiaout.today/)

== Installation ==

= Installation =

1. Upload ‘india-out-today’ folder to the ‘/wp-content/plugins/’ directory.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
